# Phycat Theme for Obsidian

一个美观、护眼、可定制的 Obsidian 主题，支持多种亮色和暗色配色方案。

## 主题特点

- **多种配色方案**：提供丰富的亮色和暗色主题选项
- **护眼设计**：低饱和度颜色，减少眼部疲劳
- **响应式布局**：适配不同屏幕尺寸
- **代码高亮**：优化的代码语法高亮
- **自定义选项**：通过 Style Settings 插件调整主题设置

## 暗色主题优化

最近对暗色主题进行了全面优化，使其更加护眼、柔和：

### 1. 现有主题改进

- **Vampire**：从高饱和度的 Dracula 风格改为低饱和度的柔和配色
- **Abyss**：从深蓝黑色改为更柔和的深蓝色调
- **Radiation**：从荧光绿色改为柔和的绿色调

### 2. 新添加的暗色主题

#### 🌙 Moonlight (Soft Purple)
- **背景色**：#2d2b57（柔和的深紫色）
- **主色调**：#c678dd（柔和的紫色）
- **特点**：优雅的紫色调，适合喜欢浪漫氛围的用户

#### 🪐 Cosmic (Deep Space)
- **背景色**：#1a1b26（深邃的太空蓝）
- **主色调**：#7aa2f7（柔和的蓝色）
- **特点**：宇宙空间感，颜色层次感强

#### 🎨 Artifact (Neutral Gray)
- **背景色**：#252526（中性深灰色）
- **主色调**：#4ec9b0（柔和的青色）
- **特点**：中性灰色调，适合专注工作

#### 🌿 Sage (Soft Green)
- **背景色**：#282c34（柔和的深灰色）
- **主色调**：#98c379（柔和的绿色）
- **特点**：自然清新的绿色调，适合长时间阅读

### 3. 共同优化

- **降低饱和度**：所有颜色都降低了饱和度，更加护眼
- **柔和发光**：发光效果透明度降低到 0.4，更加自然
- **统一结构**：保持了与原有主题一致的结构和变量命名
- **协调配色**：每个主题内部的颜色相互协调，整体感强
- **代码高亮**：为每个主题优化了代码语法高亮配色

## 亮色主题

提供多种亮色主题选项：
- **Sakura**：粉色系（默认）
- **Mint**：薄荷蓝色系
- **Sky**：清澈蓝色系
- **Forest**：自然绿色系
- **Mauve**：兰花紫色系
- **Golden Hour**：暖琥珀色系
- **Cheery**：活力红色系
- **Prussian**：深蓝色系

## 安装方法

1. 下载主题文件
2. 将文件放入 Obsidian 的主题文件夹中
3. 在 Obsidian 设置中选择 Phycat 主题
4. 使用 Style Settings 插件调整主题设置

## 自定义选项

通过 Style Settings 插件可以调整：
- 配色方案（亮色/暗色）
- 字体大小
- 间距
- 其他视觉效果
